Notas de Lançamento


2.3.2 (14/08/2022)


  • O "DNS atual" além de checar as configurações manuais agora verifica o DHCP e Gateway em último caso
  • Retornar a DNS ao padrão agora retorna as configurações reais ao invés de somente apagar as manuais (Evitar problemas onde houve configuração de IP manual)
  • A aba "Avançado" agora mantem o "Otimizar" no menu como selecionado
  • Corrigido um problema ao verificar NAT em conexões respondendo por IPV6
  • Corrigido uma falha de leitura em sistemas 64 bits
  • Corrigido um problema de resposta no botão de "Aplicar" na aba "Avançado"


    • 2.3.1 (10/08/2022)


  • Função de verificar NAT aprimorada
  • Melhorado o desempenho de diversos processos evitando a repetição de chamadas desnecessárias
  • Corrigido um atraso na mensagem de "aguarde" na função de verificar portas


    • 2.3.0 (03/08/2022)


  • Adicionado o menu de função Portas
  • Adicionado a função de testar portas TCP e UDP
  • Adicionado a função de verificar o tipo de NAT (BETA)
  • É possivel abrir a página do router ou copiar o IP Local e MAC através do programa para facilitar a abertura de portas
  • Tutoriais e artigos referentes a abertura de portas foram adicionados ao topo do novo menu
  • Agora o programa vai avisar na inicialização quando houver uma nova versão disponivel!
  • Otimização de código e arquivos internos, o programa ficou mais leve (tamanho em disco)
  • Botão de aplicar na aba de configurações avançadas agora apresenta mensagem de aguarde enquanto o processo esta rodando
  • Informações do "Sobre" agora são apresentadas em uma aba própria de forma mais detalhada com os devidos agradecimentos ♥
  • Ordem de apresentação dos elementos do menu de ping alterada para maior fluidez de utilização
  • Corrigido um problema em que a logo não era apresentada no tamanho correto em algumas circunstâncias
  • Correção/Padronização da fonte usada em alguns textos
  • Pequenas revisões textuais e de posicionamento


    • 2.2.1 (14/07/2022)


  • Adicionado um botão para voltar o DNS ao padrão do seu provedor


    • 2.2.0 (07/07/2022)


  • Adicionado o menu de função Avançado
  • Ao clicar neste menu serão exibidas as suas configurações atuais
  • É possivel escolher manualmente que alterações o programa vai realizar
  • Melhoria no processo de verificar se as configurações estão aplicadas
  • Mudanças no botão de sobre: layout e posicionamento
  • Primeira versão lançada após o site oficial


    • 2.1.0 (16/06/2022)


  • Adicionado o menu de função DNS
  • Ao clicar neste menu vai ser testado o seu ping para os 4 servidores recomendados (Cloudflare, Google, OpenDNS e Quad9)
  • Ao clicar em uma das 4 recomendações os campos serão configurados de acordo
  • É possivel Aplicar qualquer provedor DNS de sua preferência configurando manualmente
  • Adicionado opção de limpar o cache DNS (FlushDNS)
  • Mudança no layout dos botões de ação para diferenciar dos botões das caixas de mensagem


    • 2.0.0 (03/03/2022)


  • Primeira versão pública
  • Interface reformulada e nome definido como PingFast
  • Adicionado animações, dicas e interações para facilitar a navegação do usuário
  • As alterações feitas pelo programa agora só se aplicam ao adaptador de rede principal afim de evitar erros
  • Adicionado uma mensagem de erro caso o usuário tente abrir o programa sem nenhum adaptador de rede ativo
  • Refeito em .Net Framework 4.8 para evitar downloads adicionais e trazer mais comodismo aos usuários
  • Inúmeras otimizações de código e desempenho


    • 1.0.0 (23/09/2021)


  • A versão inicial não veio ao público geral, ela foi o TCC do autor e continha uma interface muito diferente e baseada no netcore 3.1
  • Aplica configurações para reduzir o tempo de resposta
  • Restaura as configurações para o padrão do Windows
  • Verifica as configurações propostas pelo otimizar para exibir a mensagem de configuração aplicada
  • Testa o ping com intervalos de 0,5s e com tempo maximo de resposta de 1000MS (pacotes maiores que isso serão considerados perda)
  • Opcionalmente é possivel salvar um log do teste de ping
  • Muitos testes foram realizados afim de corrigir bugs mas já era estável e cumpria com as propostas principais do programa por isso o nome de 1.0
