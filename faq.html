Perguntas Frequentes



Quais os requisitos mínimos?
Windows 7+ 32 ou 64 bits, com .NET Framework 4.8 instalado, os requisitos de hardware são baixos a ponto de não serem relevantes.

Preciso pagar algo para usar o programa?
Não, ele é totalmente gratuito.

Quais alterações o botão de otimizar realiza?
São muitas, todas elas podem ser vistas na parte avançada do programa.

Se eu não gostar dos resultados das alterações é fácil voltar ao padrão do Windows?
Sim, tanto as otimizações como a troca de DNS podem ser restauradas facilmente com um click através do programa.

Existe uma data especifica para futuras atualizações?
Não, este não é o único projeto no qual eu trabalho no momento então eu não posso prometer datas.

As alterações do "otimizar" são permanentes ou devo verificar após um certo tempo?
As grandes atualizações do Windows (21H1 para 21H2 por exemplo) podem reverter algumas alterações, o programa indica se as alterações ainda estão em vigor.

Existe diferença entre teste de ping pelo prompt do Windows e pelo PingFast?
O PingFast testa o ping com intervalos de 0,5s e tempo máximo de resposta em 1000ms os valores do Windows são 1s e 4000MS respectivamente

Verificar atualizações pelo programa faz parte dos planos?
A partir da versão 2.3+ o programa passou a verificar atualizações na inicialização, uma opção mais robusta e customizavel vai estar disponível no futuro

Meu Antivírus disse que o programa não é confiável, por quê disso?
Antivírus usam sistema de reputação, como o programa é novo e ainda não é popular o antivírus te dá um alerta para prosseguir com precaução o que é totalmente diferente de um alerta de vírus, coisa que o programa não contém de maneira nenhuma.

Meu Ping é baixíssimo vou ter algum beneficio em usar este programa?
O PingFast reúne soluções de rede simplificadas, você ainda pode usufruir de opções para corrigir instabilidade e conectividade.

Sinto que minha conexão está melhor, mas por quê meu ping continua alto para o meu jogo favorito?
Rede é algo complexo, você pode ter um problema de rota, se as soluções aqui propostas não te ajudarem converse com seu provedor.

Por quê o teste de portas do PingFast da resultados diferentes dos testes online?
Os testes online não tem a capacidade de abrir a conexão corretamente, você pode confiar nos resultados do PingFast.

Qual a diferença entre um teste de portas online e o teste do PingFast?
Além do resultado mais preciso, também é possível testar portas UDP.

Quem é o autor do PingFast?
Cleiton Paes. Bacharel em Sistema da Informção, conhecido como CleitonVega ou -Cleiton- na comunidade de alguns jogos.

De onde surgiu a ideia para o programa?
Dos muitos problemas de conexão reportados pelas comunidades de jogos que eu participo.

Como faço para entrar em contato com o autor?
Use o campo de contato ou envie um e-mail para suporte@pingfast.com.br.

A interface do otimizar é legal mas as outras pecam um pouco, pretende mudar?
Sim, depois que o programa contar com todas as funções planejadas inicialmente vai ser dado mais enfase na interface.

Um menu de configurações está planejado?
Sim, quando o programa estiver mais maduro novas opções de customização vão ser trabalhadas.

Ao clicar no Gateway do meu Router a configuração não foi aberta, o que aconteceu?
Muito provavelmente o acesso as configurações foi bloqueado pelo seu provedor entre em contato com ele para mais informações.

Este tal de NAT não varia de servidor para servidor?
Sim, mas através dele você pode ter uma base para identificar problemas de conexão.

As alterações deste programa são válidas para todos os adaptadores de rede?
Não, o programa realiza um teste ao iniciar identificando o seu adaptador de rede em uso.

Uso dois adaptadores de rede, as vezes cabo e as vezes Wi-Fi como prossigo?
O ideal seria usar o "otimizar" duas vezes, uma através do cabo e outra pelo wi-fi, o programa sempre vai usar apenas a rede ativa.

É planejado uma opção para alterar todos os adaptadores de rede de uma vez?
Sim, um menu de configurações esta planejado para a versão 3.0, sem data prevista ainda.

Depois de usar este programa minha conexão Wi-Fi vai ficar igual a uma cabeada?
Definitivamente não, se você joga online qualquer coisa que não seja baseada em turnos é altamente recomendado o uso de conexão cabeada.
